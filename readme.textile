h5. to get to normal workflow of edit, revert, cache, commit, and push to remote.

# (edit files)
# checkout to revert changes, e.g. @git checkout .@
# insert changes to cache (also called index), e.g. @git add src/Puzzles.cs@
# remove changes out of cache (optional), e.g. @git reset src/puzzles.java@
# diff --cached before commit, e.g. @git diff --cached@ 
# commit changes, e.g. @git commit -m 'added readme.textile.'@
# push to remote git repo, e.g. @git push git(at)github.com:0ishi/puzzles.git@
# pull from remote git repo (optional), e.g. @git pull@

h5. to shortcut common git commands using your favorite shell aliases.
